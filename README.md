## dependencies
* Vite
* Vue3
* typescript
* tailwind css

## Build Setup

```bash
# install pnpm
npm install pnpm -g

# install dependencies
pnpm install

# serve with hot reload at localhost:8080
pnpm dev

# build for production with minification
pnpm build

```
